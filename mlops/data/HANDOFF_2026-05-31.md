## 3. 오늘 작성한 파일

### `mlops/scripts/seed_activation_pct.py` (신규)

Gemini knowledge 기반으로 `exercise_muscles.activation_pct`를 일괄 시딩하는 일회성 스크립트.

**동작 방식:**
```
exercise_muscles WHERE activation_pct IS NULL 조회
    → 운동별 근육 목록 그룹핑
    → Gemini에 EMG 활성도 % 요청 (운동 1개당 API 1회 호출)
    → UPDATE exercise_muscles SET activation_pct = ?
```

**실행 조건:**
- `mlops/.env`에 `GEMINI_API_KEY` (기존 설정)
- `mlops/.env`에 `DATABASE_URL` 추가 필요

```
DATABASE_URL=postgresql://postgres:postgres@localhost:5432/scifiitsync
```
(로컬 Docker 기준. 프로덕션은 Supabase 연결 문자열 사용)

**실행 방법:**
```bash
# mlops 가상환경 활성화 후
pip install asyncpg          # 최초 1회
python mlops/scripts/seed_activation_pct.py
```

예상 소요 시간: 2~3분 (최대 47회 Gemini 호출)
멱등성: `WHERE activation_pct IS NULL` 조건으로 이미 채워진 행 skip

### `mlops/requirements.txt` (수정)

```
asyncpg>=0.30.0  ← 시딩 스크립트 DB 연결용 추가
```
